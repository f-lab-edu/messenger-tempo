# messenger-tempo
Spring 기반 채팅 메신저 API


## 구현할 기능 리스트
- 1:1 채팅, 그룹 채팅
    - 텍스트 메시지
    - 상대방이 로그아웃 상태일때도 메시지 전송
    - 로그인할때 이전에 받은 메시지 불러오기
    - 읽지 않은 유저 수 카운트
- 사용자의 접속 상태 표시 기능


- 회원 가입
- 로그인/로그아웃
- 사용자 프로필 표시
